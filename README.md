# 🧮 Calculadora em Java

Este é um projeto simples de **calculadora em Java**, desenvolvido com o objetivo de praticar a estrutura de controle `switch` e operações matemáticas básicas.

## ✅ Funcionalidades

- ✔️ Soma
- ✔️ Subtração
- ✔️ Multiplicação
- ✔️ Divisão
- ✔️ Potenciação
- ✔️ Radiciação (Exponenciação / Raiz quadrada)

## 🚀 Tecnologias Utilizadas

- Java (JDK 8 ou superior)
- IDE recomendada: Eclipse 
- Scanner para entrada de dados
